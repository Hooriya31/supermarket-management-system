# Supermarket Management System

A console-based Inventory & Sales Management System built in Java, designed to streamline business operations through automated product tracking, customer management, and real-time sales analytics.

## Problem Statement
Small to medium-sized businesses often struggle with:
- Manual inventory tracking leading to stock discrepancies
- Inefficient customer data management
- Lack of real-time sales insights
- Time-consuming manual report generation
- Poor data persistence and recovery mechanisms

## Features
- **Automated Inventory Management** – Real-time stock tracking with low-stock alerts
- **Customer Relationship Management** – Complete customer lifecycle management
- **Sales Processing Engine** – Efficient cart-based checkout system
- **Advanced Analytics** – Comprehensive reporting and business intelligence
- **Data Persistence** – File-based storage with exception handling
- **Secure Authentication** – Role-based access control system

## Tech Stack
- Language: Java
- Storage: File I/O (`.txt` files)
- Type: Console-based application

## How to Run
1. Open the project in any Java IDE (IntelliJ, Eclipse, VS Code)
2. Navigate to `Final/SMMS.java`
3. Compile and run
